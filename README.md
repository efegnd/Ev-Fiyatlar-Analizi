# Housing Prices Exploratory Data Analysis (EDA)

This project was developed as an assignment for the Skylab Computer Science Club Python & Data Science Bootcamp. It focuses on conducting an Exploratory Data Analysis (EDA) on a housing dataset to identify key variables influencing house prices.

## 🚀 Key Features & Capabilities
- **Data Integrity Check:** Handled structured data validation and missing value inspection using `pandas`.
- **Distribution Analysis:** Visualized the frequency distribution of continuous and discrete variables (area, bedrooms, stories) using `matplotlib` subplots.
- **Statistical Breakdown:** Computed average price metrics based on categorical features using Pandas `groupby` aggregation.
- **Correlation Mapping:** Generated a Pearson Correlation Coefficient matrix and visualized it with a `seaborn` heatmap to understand collinearity and assist in potential feature selection.

## 🛠️ Tech Stack & Libraries
- **Language:** Python
- **Data Manipulation:** Pandas
- **Data Visualization:** Matplotlib, Seaborn

## 📈 Sample Analytics & Insights
- **Pearson Correlation:** Used statistical measures to find the linear relationship between house features and market prices.
- **Density Grouping:** Plotted price density conditioned on room and story counts to map valuation trends.
